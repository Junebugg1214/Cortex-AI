# X Post Draft: Dev Invite (CLI)

Developers: if you want to build on a portable AI memory/identity layer, Cortex CLI is ready.

- Self-host first
- Policy-scoped sharing (`professional`, `technical`, etc.)
- Connector + graph workflow
- API + Webapp on top

Repo:
https://github.com/Junebugg1214/Cortex-AI

Quick start:

```bash
pip install cortex-identity
cortex extract <export-file> -o context.json
cortex serve context.json --enable-webapp --port 8421
```
